Useful Tools for Node and JS

Postman

Postman is a HTTP request client often used for API testing

GitHub

GitHub is a code hosting platform for version control using Git

VSCode

Visual Studio Code is a code editor for development

Node.js

Node.js is a JavaScript runtime environment and is perfect for a fullstack project

Express

Express is a web framework for Node.js

Nodemon

Nodemon is useful for development for node.js applications because it enables automatic restarting upon changes
