A couple of useful terminal commands

Basic Computer Commands
Change Current Directory 
$ cd [/path || ..]
Changes current directory
List computer files 
$ ls
Lists current directory's files
Who are you 
$ whoami
Provides the path for the current user
Node related Commands
Run a JS file 
$ node awesome.js
Runs the awesome.js file
Run a JS file with Nodemon 
$ nodemon.cmd awesome.js
Runs the awesome.js file continuously
on change
Run a script with npm 
$ npm run dev-start
runs the dev-start script from the
package.json file
Initialize a brand new Node project 
$ npm init || $ npm init -y
Setup the package.json file with 
Node package manager
Install express to the project 
$ npm i express
Setup the package.json file with 
Node package manager
Git related Commands
Initialize a new git repository from a local directory 
$ git init .
Initializes a repository for all files 
in the current directory
Stage files for a commit 
$ git add -A
Add's all files from current
directory to the repository
Commit a change 
$ git commit -m "message here"
Commmits staged files
Push all local changes 
$ git push
Pushes local changes 
to remote repository
Pull from a remote repository 
$ git pull
Pulls files from a remote repository
into current local directory
Clone a remote directory 
$ git clone https://github.com/awesomeProject
Clones a remote repository
into a local diractory
